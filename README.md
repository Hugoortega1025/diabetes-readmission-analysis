# 🏥 Diabetes Hospital Readmission Analysis

## Overview
SQL and Python analysis of 101,766 diabetic patient encounters across 
130 US hospitals (1999-2008) to identify key indicators of early readmission within 30 days.

## Tools & Platform
- **Platform:** Databricks
- **Languages:** SQL (EDA), Python (modeling - in progress)
- **Dataset:** UCI ML Repository — Diabetes 130-US Hospitals

## Key Findings
- Patients aged 20-30 show the highest early readmission rate (14.2%)
- Patient complexity (medications, diagnoses) is a stronger predictor than demographics
- Two distinct high risk profiles identified — young patients and older high complexity patients
